## 三、如何打开系统

### 方法一：使用启动脚本（推荐）

1. 进入项目目录：`c:\Users\程书岑\Desktop\实验室预约管理系统`
2. 双击运行 `启动系统.bat`
3. 等待系统自动启动完成（约1-2分钟）
4. 浏览器会自动打开 http://localhost:80/

### 方法二：手动启动

#### 步骤1：启动Redis服务
```bash
net start Redis
```

#### 步骤2：启动后端服务
```bash
# 打开命令行，进入项目目录
cd c:\Users\程书岑\Desktop\实验室预约管理系统

# 设置Java环境
set JAVA_HOME=C:\Program Files\Eclipse Adoptium\jdk-8.0.482.8-hotspot

# 启动后端
java -jar ruoyi-admin\target\ruoyi-admin.jar
```

#### 步骤3：启动前端服务
```bash
# 打开新的命令行，进入前端目录
cd c:\Users\程书岑\Desktop\实验室预约管理系统\ruoyi-ui

# 启动前端
set NODE_OPTIONS=--openssl-legacy-provider
npm run dev
```

#### 步骤4：访问系统
打开浏览器，访问 http://localhost:80/

## 四、登录系统

1. 打开浏览器，访问 http://localhost:80/
2. 在登录页面输入用户名和密码
3. 输入验证码（系统已启用验证码功能）
4. 点击"登录"按钮

## 五、功能说明

### 管理员功能 (admin)
- **实验室管理**：添加、修改、删除实验室信息
- **在线预约**：查看所有预约申请
- **预约审批**：审批教师和学生的预约申请
- **教学占用**：管理教学时间段占用
- **设备报修**：查看和管理所有报修工单
- **维修管理**：处理报修工单，更新维修状态
- **用户管理**：管理系统用户账号

### 教师功能 (teacher123)
- **实验室管理**：查看实验室信息
- **在线预约**：提交实验室预约申请
- **我的预约**：查看个人预约记录
- **教学占用**：管理教学时间段占用
- **设备报修**：提交设备故障报告
- **我的报修**：查看个人报修记录

### 学生功能 (student123)
- **实验室管理**：查看实验室信息
- **在线预约**：提交实验室预约申请
- **我的预约**：查看个人预约记录
- **设备报修**：提交设备故障报告
- **我的报修**：查看个人报修记录

## 七、常见问题

### Q1: 无法访问系统怎么办？
A: 检查Redis服务、后端服务、前端服务是否正常启动。

### Q2: 忘记密码怎么办？
A: 联系管理员重置密码，或使用其他账号登录。

### Q3: 验证码看不清怎么办？
A: 点击验证码图片可以刷新验证码。

### Q4: 预约被拒绝怎么办？
A: 联系管理员了解原因，或重新提交预约申请。

## 附录：启动脚本使用说明

### 启动脚本文件
- 文件名：`启动系统.bat`
- 位置：项目根目录 `c:\Users\程书岑\Desktop\实验室预约管理系统\`

### 使用方法
1. 双击运行 `启动系统.bat`
2. 脚本会自动执行以下操作：
   - 检查Java环境
   - 启动Redis服务
   - 启动后端服务（等待15秒）
   - 启动前端服务（等待30秒）
3. 启动完成后，按任意键打开浏览器

### 注意事项
- 首次启动可能需要等待1-2分钟
- 如果端口被占用，脚本会提示错误
- 关闭脚本窗口不会停止服务，需要手动关闭Java和Node进程
